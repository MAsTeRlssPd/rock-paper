# rock-paper
A project by Odin Project <br>
This project is a console-based Rock, Paper, Scissors game built as part of The Odin Project’s Foundations curriculum. The focus of this assignment was to apply fundamental JavaScript programming principles such as variables, conditional logic, case-insensitive string manipulation, and functional decomposition to break down and solve a multi-step problem. The final program successfully handles user input, generates randomized computer selections, tracks running scores over a five-round match, and outputs the results directly to the browser console.
